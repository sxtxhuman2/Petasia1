# Petasia1

대만 시장에 한국산 반려동물 제품 판매를 준비하기 위한 **시장 분석 + 상품/업체 평가 리포트 웹앱**입니다.

## 구성

- `index.html`: GitHub Pages용 단일 페이지 앱
- `styles.css`: 반응형 레이아웃/카드/UI 스타일
- `app.js`: 필터, 체크리스트, 점수계산, 상태 저장(localStorage)
- `data/vendors.json`: 국내 이커머스 기반 조사 템플릿 데이터(100개)

## 사용 방법

1. 저장소를 GitHub에 푸시
2. Settings → Pages → Branch를 `main` / root로 설정
3. 배포된 URL 접속 후
   - 시장별(대만/한국), 가격대(프리미엄/중가/저가), 카테고리 필터
   - 후보 체크 및 점수 입력
   - 최종 우선순위 의사결정

## 주의

- 본 데이터는 초기 리서치/평가를 위한 실무 템플릿입니다.
- 실제 수입/통관/인증(FDA, 동물용품 규정 등)과 현지 판매 가능성은 반드시 별도 검증하세요.
